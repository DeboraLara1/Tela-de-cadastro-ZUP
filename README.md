# Desafio Front-End Zup


## Pré-requisitos
- Angular 10.0.0
- Npm
- Nodejs

## Passo a passo

1 - Clone este repositório;
<br /> <br />
2 - Instalar pacote de dependências para fazer build do projeto;
<br /> `npm install`
<br /> <br />
3 - Executar/subir API Json-server;
<br /> `json-server --watch db.json`
4 - Executar/subir projeto;
<br /> `npm start`
